Introduction

This Page contains Instructions on how to compile my GameMaker Studio 2 projects (all my Pizza Tower Ports and some others) to target the Web Browser

This DOES NOT use HTML Export, as it is quite frankly, terrible

Instead, my method uses the GX.GAMES Export, which has more support and works way better than HTML

Also, I strongly recommend using the GX.GAMES export rather than the HTML export in any GMS2 project

The GX.GAMES export option is actually a WebAssembly Export in disgiuse. Why YoYo Games hides that and makes your project hosted on only GX.GAMES is unknown, but it's probably to force developers that want to port their games to the Web to use their service instead.

But this guide will help you break those restrictions for any GameMaker Studio 2 Project!
